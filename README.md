## Hasil yang bakal lo liat

**Percaya ga percaya, sekalipun lo dapet jackpot gede berkali-kali, ujung-ujungnya saldo bakal habis.**

Coba aja langsung buka filenya:
1. Buka file `index.html` di browser
2. Pilih taruhan kecil dulu (10rb)
3. Klik gacha berkali-kali
4. Amati polanya: jackpot dikit, terus disusul denda gede
5. Kalo saldo udah gede, taruhan bakal naik sendiri (lo gak sadar kalo lo mulai main lebih gede)

Sampe sini lo pasti ngerti: **Sistem judi online tuh gak pernah dirancang buat bikin lo kaya.**
